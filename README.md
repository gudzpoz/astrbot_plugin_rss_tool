# astrbot-plugin-rss-tool

为 AstrBot 添加 RSS 技能，让你的 AstrBot 可以自主获取、筛选 Atom/RSS 信息。
建议配合（目前实验性的）[AstrBot 主动型能力](https://docs.astrbot.app/use/proactive-agent.html)
使用。例如，你可以让 Agent “每天早上 8 点获取一次 RSS 并筛选 XX 领域的新闻”。
